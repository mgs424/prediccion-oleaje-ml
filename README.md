# Análisis y Predicción de Oleaje con Machine Learning

Este proyecto tiene como objetivo analizar patrones climáticos y construir un modelo capaz de predecir eventos de oleaje fuerte, 
a partir de datos reales (altura de ola, periodo, dirección y temperatura superficial).

1. Limpieza de Datos: Tratamiento de fechas y eliminación de errores de sensores.
2. Análisis Exploratorio: Estudio de correlaciones y detección de patrones ocultos mediante técnicas no supervisadas (PCA y K-Means).
3. Modelado Predictivo: Entrenamiento y comparación de un Árbol de Decisión frente a un Random Forest para clasificar el estado del mar.
